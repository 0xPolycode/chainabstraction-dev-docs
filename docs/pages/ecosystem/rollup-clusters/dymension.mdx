# Dymension

Dymension is a decentralized network that connects RollApps to users, providing security, bridging, and liquidity. RollApps are scalable, consensus-free blockchains built using the RollApp Development Kit (RDK), designed for various applications like games, DeFi, and NFT projects.

## Overview

Dymension serves as a hub for RollApps, offering a unified platform for deploying and interacting with application-specific rollups. It simplifies the process of creating and managing blockchain applications while ensuring scalability and customizability.

## Benefits

- **Scalability**: RollApps are designed for high performance and throughput
- **Customizability**: Developers can create tailored blockchain applications
- **Security**: Leverages Dymension's validator network for RollApp security
- **Interoperability**: Facilitates easy communication between RollApps via IBC
- **Liquidity**: Provides shared liquidity through an embedded AMM

## Key Features

### RollApps

- Custom token creation and gas fee management
- On-chain governance mechanisms
- IBC bridging for ecosystem access

### Dymension Hub

- Decentralized Delegated Proof-of-Stake L1 blockchain
- Secured by the `DYM` token
- Embedded Automated Market Maker (AMM) for shared liquidity

### RollApp Development Kit (RDK)

Pre-packaged set of modules for common blockchain functionalities

### RollApp Virtual Machine (RVM)

Novel dispute-resolution mechanism for supporting various execution environments

### Inter-Blockchain Communication (IBC)

Enables safe message transferring between Dymension RollApps
