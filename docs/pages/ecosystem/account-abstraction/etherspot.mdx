# What is Etherspot?

Etherspot is a multi-chain Account Abstraction platform that empowers developers to build seamless Web3 user experiences. Designed for dApps, wallets, games, and EVM-compatible rollups or L1/L2 chains, Etherspot removes common usability pain points, enabling frictionless interaction for Web3 users across multiple blockchains.

## Key Features

- **Comprehensive Account Abstraction Infrastructure**: Fully compatible with ERC-4337 and ERC-7579 standards.
- **Seamless Integration**: Features a developer-friendly React library, TransactionKit, for easy implementation.
- **Multi-Chain Support**: Compatible with over 22 EVM chains.
- **Open-Source**: Transparent and accessible for community collaboration and enhancement.

## What Etherspot offers?

- [Etherspot Prime ERC-4337 SDK](https://etherspot.fyi/prime-sdk/intro): Designed to deliver a seamless Web3 UX.
- [Etherspot Modular ERC-7579 SDK](https://etherspot.fyi/modular-sdk/intro): Makes smart accounts customizable, allowing developers to enhance security and user experience by integrating various modules.
- [Skandha Bundler](https://etherspot.fyi/skandha/intro): Batches multiple transactions into one, streamlining interactions and improving efficiency. Skandha has MEV protection and supports Shared Mempool on Polygon.
- [Arka Paymaster](https://etherspot.fyi/arka/intro): Provides sponsored and gasless transactions, simplifying the transaction process.
